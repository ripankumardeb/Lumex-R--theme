# Changelog

## [1.0.0] — 2024-01-01

### Added
- Lumex Dark and Lumex Light editor themes with full token + UI coverage
- 160+ SVG file and folder icons (Lumex Icons theme)
- Live Customizer webview panel (status bar button + Command Palette)
- 14-slot color palette editor — accepts hex, rgb(), hsl(), CSS names, Tailwind tokens
- File icon uploader — replace any icon with PNG or JPG (max 512 KB)
- 6 built-in preset themes: Lumex Dark, Lumex Light, Midnight Ocean, Sakura, Nord Drift, Cyberpunk Neo
- One-click Restore Defaults with confirmation dialog
- Persistent custom icon storage via VS Code globalState
- Color validation with inline error tooltip
- Live code preview panel with Prism.js syntax highlighting
